# Banana Coffee Advisor Android App

This is a simple Android app prototype for early detection and advisory support for banana bacterial wilt and coffee leaf rust in Bushenyi district.

## How to open
1. Open Android Studio.
2. Choose `Open` and select the `banana-detection-system/android-app` folder.
3. Sync the Gradle project and run the app on an emulator or Android device.

## Functionality
- Choose between banana and coffee.
- Select symptoms observed in the field.
- Get instant advisory guidance.

## Notes
- This app uses local symptom logic and does not yet connect to a remote backend.
- For data sharing, extend the app to send reports to the backend at `http://localhost:4000/report`.
